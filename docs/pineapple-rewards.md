# Pineapple Rewards System

## Overview

Pineapples (ğŸ) are the gamified currency in Vamo that reward consistent daily progress. They are earned through daily logins, uploading evidence, and achieving streak milestones. Pineapples can be spent to reveal potential customers in the "Find Customers" feature.

## Earning Pineapples

There are three ways to earn pineapples:

1. **Daily Login Bonus** - Automatic credits for logging in each day (scales with streak)
2. **Evidence Credits** - Bonus credits for uploading evidence (diminishing returns per day)
3. **Milestone Bonuses** - One-time rewards for achieving major streak milestones
